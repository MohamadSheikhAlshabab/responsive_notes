# Responsive Image :

when you use inline width & height in image you improve SEO of your website.
so with this way you make the image responsive and accomplish better SEO.

<pre>
<code>
<style>
  img {
    width: 100%;
    height: auto;
    aspect-ratio: attr(width) / attr(height);
  }
</style>
</code>
